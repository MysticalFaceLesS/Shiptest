### Описание мода

Устраиваем великий рефакторин оутфитов.

Оутфиты Нанотрейзена: Добавляем отсутствующий оутфит
Оутфиты солфеда: Этот мод добавляет стафф одежду для солфеда.
Оутфиты Интека: Этот мод включает различные вещи и прочий код для Интек.
Оутфиты Гамма фракции: Этот мод добавляет вещи для Гаммы.
Оутфиты Элизиума: Этот мод вносит вещи для элизиума.
